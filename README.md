# INCREMENTAL SEARCH BAR

## Description:

Incremental search bar that allows the user to search instantaneously among a list of world countries.

## Developed with:

- HTML
- JavaScript/Jquery
- CSS
- Ajax requests

## Features:

Incremental search responsive to typing and deleting
